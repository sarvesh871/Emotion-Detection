# Project Name
Emotion Recognition Dashboard
---

# Short Description
A platform to recognise ones emotions. Works using Amazon services like amazon lambda, amazon rekognition, etc.
---

# AWS Services
Amazon S3 (X2, one for hosting and the other one for storage)
Amazon CloudFront
Amazon Lambda
Amazon CloudWatch
Amazon API Gateway
Amazon Rekognition
---

# Frontend
HTML
CSS
JavaScript
---

# Backend
Amazon Lambda
Python
---

# Deployment
Deployed using S3 and CloudFront for camera access as modern browsers require HTTPS links for permissions such as camera, microphone, etc.
---

# API Endpoints
HTTP API using routes /happiest (GET), /saddest (GET), /recent (GET), /latest (GET), /upload-url (GET)
